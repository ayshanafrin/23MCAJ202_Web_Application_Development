Welcome to the exploration of 3 different kinds of Styles......!

CSS provides three different ways to style a webpage: Inline, Internal, and External CSS. Each method has its unique use case and impact on design and maintainability.

INLINE STYLE SHEET

Inline CSS is applied directly to an element using the style attribute.

INTERNAL STYLE SHEET

Internal CSS is written inside the style tag within the head section.

EXTERNAL STYLE SHEET

External CSS is stored in a separate file and linked to the HTML document using the <link> tag.
